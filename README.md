# Embedded Systems Interview Lab

This repository is a structured, interview-focused embedded systems lab
demonstrating deep understanding of firmware engineering concepts.

## Topics Covered
- Embedded C internals
- ARM Cortex-M bare-metal programming
- FreeRTOS internals
- Memory & linker optimization
- Bootloader design
- Peripheral driver architecture
- Debugging techniques
- Embedded system design patterns

## How to Navigate
Each numbered folder inside this repository represents a self-contained
module with theory, code examples, diagrams, and interview Q&A.

## Target Platforms
- STM32 (Cortex-M3 / M4)
- Renode / QEMU
